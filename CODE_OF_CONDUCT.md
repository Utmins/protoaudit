# Code of Conduct

ProtoAudit aims to maintain a welcoming and collaborative open-source environment.

## Expected behavior

Participants in the project community should:

- be respectful and constructive
- focus discussions on technical improvement
- welcome new contributors
- avoid personal attacks or hostile language

## Unacceptable behavior

- harassment or discrimination
- abusive language
- disruptive conduct in issues or pull requests

## Enforcement

Project maintainers may remove or moderate contributions that violate these guidelines.
