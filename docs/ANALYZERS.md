# Analyzers

## Protocol analyzer

The protocol analyzer now includes:
- challenge-like field extraction
- repeated response detection
- message state tracking
- protocol phase inference
- handshake pattern detection
- repeated phase loop detection

## Crypto analyzer

The crypto analyzer normalizes metadata such as proof of possession, participant binding, domain separation, and trust-boundary assumptions.

## Randomness analyzer

The randomness analyzer reports repeated output ratios, uniqueness, common prefix bits, truncation exposure, and state-recovery feasibility.
